# North Carolina Energy Analysis

## Overview

This project collects, processes, and analyzes electricity demand and historical weather data for North Carolina to understand better how weather patterns influence energy consumption. The project combines data engineering, exploratory data analysis, and data visualization to uncover trends and build an interactive energy dashboard.
